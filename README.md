# ProShop

Proshop is an innovative and user-friendly eCommerce website created using MERN stack (MongoDB, ExpressJS, ReactJS and NodeJS,
Bootstrap is used for responsive design across different devices.
It currently has cart functionality, View Product list, Single Product Page, Cart Page etc.
